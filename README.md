## 🧩 Loyiha Texnik Imkoniyatlari

Ushbu modul orqali quyidagi ilg'or Android texnologiyalari amalga oshirilgan:

*   🎯 **Bottom Sheet Mastery:** Ma'lumotlarni kiritish uchun qulay va zamonaviy `ModalBottomSheet` komponenti.
*   🖼️ **Custom Dialog Architecture:** Foydalanuvchi bilan muloqot uchun maxsus dizayndagi `AlertDialog` va `MaterialDialog`.
*   ⚡ **Action Snackbar Integration:** Xabarlarni ko'rsatish va amallarni qaytarish (`Undo`) tugmasi bilan ishlash.
*   🧠 **Smart Quiz Engine:** Bilimlarni tekshirish uchun `Green/Red` indikatorli interaktiv test tizimi.
*   📊 **Local Data Persistence:** Natijalarni va foydalanuvchi ma'lumotlarini `SharedPreferences` orqali saqlash.
